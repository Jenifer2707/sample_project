Front-End Developement

Front-end development is the process of building components that interact with users. Examples are the user interface, buttons, user-entered data, websites, and user experience (UX) features.A Front-End Developer is responsible for developing new user-facing features, determining the structure and design of web pages, building reusable codes, optimizing page loading times, and using a variety of markup languages to create the web pages.

Front-End Languages:

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • ReactJs

These are the languages that are used in the front end development.
